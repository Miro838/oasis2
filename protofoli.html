My Projects

Slack landing page

The Slack landing page project features a modern and engaging design aimed at promoting Slak, a team collaboration tool. It includes a header with navigation links and a sign-up button, a hero section with a welcome message and key features, a features section detailing core functionalities like real-time messaging and task management, a testimonials section showcasing user feedback, and a contact form for inquiries. The footer provides social media links and copyright information, while a modal offers an additional sign-up prompt. The page uses responsive design principles and Font Awesome icons to enhance visual appeal and usability.

Temperature Converter

The Temperature Converter project is a web-based application that allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin. The interface features a clean design with an input field for temperature entry , a dropdown menu for selecting the temperature unit, and a "Convert" button to trigger the conversion. The application displays the current time and provides real-time temperature conversion results in the selected unit. It includes functionality for updating the time every second and calculates and displays the equivalent temperatures in other units based on user input.
